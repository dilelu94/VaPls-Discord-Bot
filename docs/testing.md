# Testing

## Current coverage
- `tests/testSoundpad.py`: navegación de categorías/subcarpetas, paginación y
  reproducción básica del Soundpad.

## Running tests
```bash
python3 -m unittest tests/testSoundpad.py
```

Notas:
- Los tests requieren dependencias de Discord (`py-cord`/`discord`).
- No hay suites adicionales automatizadas en este repositorio.
